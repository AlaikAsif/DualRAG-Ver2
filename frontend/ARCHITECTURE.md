# Frontend Architecture

## Overview
The `frontend/` module contains the web UI for the chatbot.

## Components

### index.html
- Landing page
- Navigation
- Introduction

### chatbot.html
- Main chat interface
- Message display
- Input handling

### static/js/
- API communication
- Chat UI logic
- Report viewing

### static/css/
- Styling
- Responsive design
- Component styles

## Technology Stack
- Vanilla JavaScript or React
- HTML5
- CSS3
- Responsive Design

## Features
- Real-time chat interface
- Report viewing
- File upload
- Response streaming (optional)
